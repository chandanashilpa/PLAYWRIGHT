# Playwright Practice

This repository contains basic JavaScript practice files for learning and experimenting with Node.js and JavaScript behavior.

## Files

- `basics.js` - Demonstrates simple functions such as string reversal and addition, along with example test assertions.
- `hoisting.js` - Contains examples related to JavaScript hoisting behavior.

## Usage

Run the files using Node.js:

```bash
node basics.js
node hoisting.js
```

## Notes

- The `basics.js` file includes a simple `reverse_string` utility and a small test function for `add`.
- The repository is set up to keep practice code organized for learning JavaScript fundamentals.
